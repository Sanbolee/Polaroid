Mac 系统下编译安装PHP Redis 扩展

1. 首先看一下是否已经安装了PHP扩展

  • $ php -m

如果显示有redis，那可以关闭这篇文章了。

没有的话，我们继续

2. 进入PHP官网下载redis扩展下载扩展

例如下载redis-5.3.1.tgz

3. 解压压缩包，然后进入redis-5.3.1文件夹

  • $ phpize && ./configure

4. 编译

  • $ make && make install

如果这里出现类似下面的错误：

  • mkdir: /usr/local/Cellar/php@7.3/7.3.13/pecl: File exists
  • mkdir: /usr/local/Cellar/php@7.3/7.3.13/pecl: No such file or directory
  • make: *** [install-modules] Error 1

按照下面的步骤可解决：

  • 打开 /usr/local/Cellar/php@7.3/7.3.13/bin/php-config 
  • extension_dir='/usr/local/Cellar/php@7.3/7.3.13/lib/20180731' 中的 『pecl』 换成 『lib』
  • 重新执行第3步

如果显示类似如下信息，则表示成功 🍎

  • Installing shared extensions: /usr/local/Cellar/php@7.3/7.3.13/lib/20180731/

5. 配置 php.ini文件，一般位置在/usr/local/etc/php/7.3/php.ini，打开添加extension="redis.so"

6. 重新执行步骤1看看是否有redis扩展了。

如果还是没有，那么看看php.ini文件里的这行extension_dir = "/usr/local/Cellar/php@7.3/7.3.13/lib/20180731"是否和上面🍎 那里的路径一样，如果不一样就改成一样即可。
