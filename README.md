## About Polaroid  
「Polaroid」一个简约的拍立得风格的博客，基于Bootstrap4 + jQuery 开发的静态版本，博客前台。  
特点：简约、大气、美观，拍立得风格。  
作用：可用于搭建个人博客。
> Vue版本的前台、后台，PHP、JAVA、Golang、Python开发的API也在提上日程中
